**Project archived**

This project has been replaced by [`tkl-installer`](https://github.com/turnkeylinux/tkl-installer)
(ti) - a custom TurnKey/GNU Linux live installer project. Some it functionality
is inspired by di/di-live, but it has been written from scratch.

di-live
=======

Installer for "Live" system (i.e. OS running from ISO/CD/USB) leveraging
components of Debian Installer (di).
